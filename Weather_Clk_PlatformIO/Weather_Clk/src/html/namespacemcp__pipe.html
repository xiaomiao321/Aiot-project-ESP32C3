Detailed Description

此脚本用于连接到MCP服务器，并通过WebSocket端点进行输入输出管道传输。
版本: 0.1.0

使用方法:

export MCP_ENDPOINT=<mcp_endpoint>  # 设置环境变量
python mcp_pipe.py <mcp_script>      # 运行脚本并指定要执行的MCP脚本
